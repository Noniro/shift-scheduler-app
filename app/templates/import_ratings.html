Import Worker Difficulty Ratings (Matrix Format)

Matrix Format Instructions:
  • The CSV has workers as rows and job roles as columns
  • Each worker fills out one row with ratings for all roles they're qualified for
  • Ratings must be integers from 1-5 (1=Easy, 5=Very Hard)
  • Leave cells empty or "N/A" for roles you can't/won't rate
  • Values will be automatically normalized (>5→5, <1→1)
  • Multiple ratings for the same role are averaged

Processing Rules:
  • Integer Only: 3.7 → 4, 2.2 → 2
  • Normalization: 6 → 5, 0 → 1
  • Empty/N/A: Ignored (worker didn't rate that role)
  • Averaging: If multiple workers rate same role: (3+4+2)÷3 = 3.0
